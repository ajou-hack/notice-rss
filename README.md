# 공지사항

아주대학교 공지사항을 [RSS 2.0 형식](https://www.rssboard.org/rss-specification)으로 제공합니다. 공지사항 데이터는 아주대학교 홈페이지의 [아주광장 > 공지사항 > 일반공지](https://www.ajou.ac.kr/kr/ajou/notice.do) 페이지를 1시간마다 스크래핑하여 업데이트하고 있습니다.
